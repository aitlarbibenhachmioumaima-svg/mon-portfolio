# 🌐 Oumaima Portfolio

Personal portfolio website built with **React + Vite + Tailwind CSS** to showcase my projects, skills, and experience .

---

## 🚀 Live Demo

👉 https://mon-portfolio-ecru-sigma.vercel.app

---

## 🛠️ Technologies Used

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS
* 🧩 Radix UI
* 🧠 TypeScript
* 🔀 React Router
* 🪄 Framer Motion

---

## 📂 Project Structure

```
src/
│── app/              # Layout & routing
│── components/       # Reusable UI components
│── data/             # Static data (projects, profile...)
│── pages/            # Pages (Home, Projects, Contact...)
│── lib/              # Utilities
```

---

## ✨ Features

* Responsive design (Mobile & Desktop)
* Dark / Light mode 🌙☀️
* Clean and modern UI
* SEO optimization (React Helmet)
* Projects showcase
* Contact section
* Download CV button
